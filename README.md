# Who is Hiring

This project fetches data from the HN firebase API and stores it locally for further analysis. The data is serialized to JSON and SQLite allowing freedom to query in SQL or JSON viewers

## Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm (v9 or higher)

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/who-is-hiring.git
    cd who-is-hiring
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Set up environment variables:**

    Create a `.env` file in the root directory based on the provided [.env](.env.example) file.

## Running the Application

1. **Start the server:**

    ```bash
    npm run start
    ```
